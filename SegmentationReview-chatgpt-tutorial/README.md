# SegmentationReview — 3D Slicer Extension

> A 3D Slicer extension for structured review and QA of medical image segmentations.

## What does this extension do?

SegmentationReview lets radiologists and researchers:

- Load and review NRRD / NIfTI segmentation files
- Accept, reject or flag individual segments with comments
- Export review summaries as JSON or CSV
- Track reviewer inter-rater agreement

## Installation

```bash
# Clone the repo
git clone https://github.com/your-org/SegmentationReview.git
cd SegmentationReview

# Install dev dependencies
pip install -r requirements-dev.txt
```
