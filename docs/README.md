# Starting off

## Downloading the software

The first step is to install Git's software. This can be done [`Here`](https://git-scm.com/install/). Git is Terminal based, meaning we can only use it in a terminal (Powershell/Command Window for Windows)
If you have little experience working in a terminallike, start off with a GUI (interface) you can also install GUI Clients [`Here`](https://git-scm.com/tools/guis).

For beginners I recommend Github Desktop because we will be using this for the examples in [GUI tutorial](./GUI_tutorial)
